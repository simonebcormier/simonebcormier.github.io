HLSL Shaders

I developed HLSL shaders in Unity, including a dynamic Hologram Shader and a stylized Waterdrop Shader, to create captivating visual effects for interactive environments. These shaders leverage animated vertex displacement, rim lighting, and transparency to produce glowing, energy-like surfaces and holographic visuals.

Project Info

  • Project: VFS Project
  • Role: Technical Artist, Shader Programmer
  • Software: Unity, HLSL/ShaderLab
  • Programming Languages: HLSL, ShaderLab, C#

Responsibilities

  • Developed a Hologram Shader using HLSL to create animated, transparent holographic effects with customizable color, rim lighting, and wave distortion, ideal for sci-fi and futuristic visuals.
  • Created the Waterdrop Shader featuring dynamic rim lighting, fresnel-based transparency, and pulsing vertex animation for stylized, glowing energy effects.
  • Implemented advanced shader properties for real-time control over color, animation speed, wave intensity, and transparency, enabling rapid iteration and artistic experimentation.

Hologram Shader Features

  • Animated horizontal wave distortion
  • Customizable hologram and rim colors
  • Adjustable transparency and scale
  • Glowing rim effect for a sci-fi look

Waterdrop Shader Features

  • Pulsing, wavy vertex animation using sine/cosine
  • Dynamic rim lighting with fresnel-based alpha
  • Emissive, glowing edge for magical or energy-based visuals
  • Real-time control over rim intensity, power, and color
