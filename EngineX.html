Custom C++ Engine - Brick Breaker

We developed Brick Breaker using a custom C++ game engine. The game features a menu scene and a main gameplay scene where the player controls a paddle to bounce a ball and break bricks. The game includes both breakable(bricks) and non-breakable boxes(the paddle), with the playerï¿½s paddle and the ball each having their own collider systems. The objective is to clear all breakable bricks to win, while avoiding letting the ball pass the paddle, which results in a loss. The game implements robust collision detection, win/lose states, and dynamic ball physics for engaging gameplay.

Project Info

  • Project: VFS Project
  • Role: Programmer
  • Software: Custom Game Engine
  • Programming Languages: C++

Responsibilities

  • Implemented responsive paddle controls using A/D keys for smooth left and right movement
  • Developed and integrated collision logic for circle/box interactions, including; Ball and paddle collisions, ball and brick collisions, ball and game border collisions
  • Programmed the ball to bounce off the paddle with direction and angle changes based on the hit position (center vs. edges), enhancing gameplay skill and challenge
  • Ensured the ball spawns with a randomized downward force, creating varied and unpredictable game starts
