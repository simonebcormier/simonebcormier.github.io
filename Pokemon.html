Pokemon C# Console Game

Project Overview

Me and one other programmer created a text-based Pokemon game to be played on the console application. Players embark on a journey where they choose a starter pokemon(Squirtle, Charmander or Bulbasaur), engage in turn-based battles against enemy pokemon and progress thorugh the story until they reach the boss battle. This project helped me practice both my object-oritented programming principles as well as collaborative programming.

Project Info

  • Project: VFS Project
  • Role: Programmer
  • Programming Languages: C#

Responsibilities

  • Object-Oriented Design
  • Implementation of inheritance with base pokemon class
  • Turn Based Combat System
  • Progression System
  • Error and user input validation
  • Console UI using ASCII art and formatting

Turn Based Combat System

  • Player and enemy pokemon take turns selecting actions(Attack/Defensive/Offensive)
  • Each attack considers type advantages(Fire/Water/Grass type pokemons) for more strategy
  • Accuracy and damage numbers are calculated with stat modifiers from offensive and defensive attacks and some randomization
  • Final boss battle will always be the opposite type of the pokemon you chose for added difficulty

Progression System

  • Player can choose which pokemon they get to use at the beginning of the game
  • Player gains experience and levels up after winning a battle
  • Player gains in game currency to spend on healing items in a shop
  • Battles become more challenging as you progress through the game
  • When level 8 reached, final battle commences
  • Player choices during battles affect the players journey and the battle outcome
