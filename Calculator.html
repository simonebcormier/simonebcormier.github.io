Android Calculator App

I designed and implemented a functional calculator built with Android Studio using Java, featuring basic arithmetic operations, decimal support, and a negative number toggle. The app includes a 5-operation history feature and phone rotation support, demonstrating my proficiency in Andorid development and Java programming.

Project Info

  • Project: Carleton Solo Project
  • Role: Programmer, Designer
  • Software: Android Studio
  • Programming Languages: Java

Responsibilities

  • Complete mathematical operations (add, subtract, multiply, divide)
  • Decimal and negative number support
  • Clear and backspace functionality
  • 5-operation history tracking
  • Responsive design with phone rotation support
