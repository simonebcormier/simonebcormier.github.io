Futuristic Crate 3D Model

A detailed sci-fi construction crate created from concept to final render using Maya. The assets features optimized UV mapping and was fully textured with PBR workflows, inlcuding albedo, metallic, roughness, and normal maps. This project demonstrates a complete 3D assets pipeline, from modeling to texturing to high-quality rendering, with a focus on game-ready standards and visual style.

Project Info

  • Project: Carleton Project
  • Role: 3D Modeler
  • Software: Maya

Responsibilities

  • Modeled the 3D crate asset in Maya
  • Created and optimized UV maps for efficient texturing
  • Applied PBR texturing techniques (albedo, metallic, roughness, normal)
  • Rendered high-quality images of the final asset
  • Ensured model was game-ready and visually consistent
