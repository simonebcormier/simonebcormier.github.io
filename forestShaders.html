HLSL Unity Forest Scene

I developed a stylized nighttime magical forest scene in Unity, utilizing a suite of custom HLSL shaders to create a captivating and immersive environment. The scene features glowing mushrooms, animated fireflies, volumetric fog, and stylized foliage, all working together to evoke a sense of wonder and enchantment. Each shader was crafted in HLSL to enhance the visual storytelling, from atmospheric depth to dynamic lighting effects, resulting in a cohesive and magical experience.

Project Info

  • Project: VFS Project
  • Role: Technical Artist, Shader Programmer
  • Software: Unity, HLSL/ShaderLab
  • Programming Languages: HLSL, ShaderLab, C#

Responsibilities

  • Developed a Fog Shader using HLSL to add volumetric fog, enhancing depth and creating a mystical nighttime atmosphere.
  • Created a Mushroom Shader in HLSL for glowing, emissive mushrooms that illuminate the scene with bioluminescent effects.
  • Implemented an Unlit Tree Shader with HLSL for stylized trees, including wind animation for natural movement and a consistent cartoon-like look.
  • Designed a Bushes Shader in HLSL to render soft, magical foliage that blends seamlessly with the environment.
  • Built a Firefly Shader using HLSL to animate glowing fireflies, utilizing emissive properties and movement to add dynamic, magical elements to the scene.
