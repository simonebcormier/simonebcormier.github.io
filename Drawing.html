Drawing App

I designed and developed a comprehensive drawing application using Android Studio and Java. The app features a full-featured drawing canvas with multiple drawing tools including free draw, line draw, square draw, and polygon draw modes. Additional features include pen size adjustment, color selection, eraser tool, undo functionality, and gesture-based controls like two-finger and three-finger drawing. The app also includes a shake-to-clear feature for quick screen erasure.

Project Info

  • Project: Carleton Project
  • Role: Programmer, Designer
  • Software: Android Studio
  • Programming Languages: Java

Responsibilities

  • Interactive drawing canvas with touch input handling
  • Multiple drawing modes (free draw, line, square, polygon)
  • Pen size slider and color selection tool
  • Undo and clear functionality
  • Eraser tool with adjustable size
  • Gesture-based drawing (two-finger, three-finger)
  • Accelerometer integration for shake-to-clear feature
